# typescript로 작성한 session 로그인 로그아웃

## 심플하게 구현한 웹 로그인 로그 아웃 코드

구현 내용은 아래와 같이 요약함

- rest api 형태로 만듬 (redis, mysql 필요)
- jest를 이용해서 rest api를 코드 검증
- swagger 이용하여 rest api 확인 가능
- swagger에서 사용하는 definitions은 자동으로 코드 생성(yaml) 기능 있음
